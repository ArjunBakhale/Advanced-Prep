# Week 8: Deploying Your Website

In Week 8 of the Nivaro AI Website Program, students will learn the essentials of version control using GitHub and how to deploy their website to the web using Vercel. This week is crucial for understanding how to manage and showcase your web development projects effectively. {{ className: 'lead' }}

## Lesson 8.1: Deploying to Vercel

This lesson guides students through the process of deploying their website to Vercel, a cloud platform for static sites and Serverless Functions.

### Setting Up Vercel

1. **Create a Vercel Account:**

   - Go to [Vercelâ€™s website](https://vercel.com/) and sign up for an account.
   - Optionally, install Vercel CLI for additional control: `npm i -g vercel`.

2. **Deploying Your Project:**

   - Connect your GitHub account to Vercel.
   - Import your GitHub repository in Vercel.
   - Follow the prompts to set up your project and deploy it.
   - Once deployed, Vercel provides a URL to view your live site.

3. **Continuous Deployment:**
   - Learn how Vercel automatically deploys your site with every push to your GitHub repository.
   - Explore features like preview deployments for pull requests.

### Resources:

- [Vercel Documentation](https://vercel.com/docs)
- [Deploying a Next.js App to Vercel](https://nextjs.org/docs/deployment) - Specific guide for deploying Next.js applications.

### End Goal

By the end of Week 8, students will have their websites version-controlled with GitHub and deployed online using Vercel. They will understand the workflow of professional web development and deployment.
